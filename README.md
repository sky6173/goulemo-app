# 购了么

“购了么”是一个本地优先的 Android 购物清单应用，用来记录日常采购、待买和已买商品。

## 下载

- 最新版本：v1.0.2
- 安装包：app-release.apk
- 下载地址：https://github.com/sky6173/goulemo-app/raw/main/app-release.apk

## v1.0.2 更新内容

本次主要加入价格统计和更清晰的商品操作入口：

- 新增商品单价记录，填写数量和单价后自动计算商品金额
- 顶部显示待买金额、已买金额和总计金额
- 商品卡片显示单个商品按数量计算后的金额
- 搜索支持商品名、分类、数量、单位、价格和多关键词匹配
- 新增智能排序、手动排序和长按拖动调整商品顺序
- 长按商品会弹出操作菜单，可直接编辑信息、进入批量选择、移动状态或删除
- 批量选择后可批量移动到已买/待买、批量删除

## v1.0.1 更新内容

本次主要优化界面动效，让操作反馈更柔和：

- 数量统计变化增加轻微弹入反馈
- 顶部进度条切换为平滑动画
- 分类卡片、筛选按钮、底部导航增加柔和选中动效
- 商品列表切换和空状态展示增加淡入上滑效果
- 已买商品状态变化增加轻微视觉过渡
- 保持原有布局和操作逻辑不变

## v1.0.0 更新内容

首个公开测试版本，当前支持：

- 添加商品名、数量、单位和分类
- 按待买、已买查看购物清单
- 搜索商品名称或分类
- 长按商品进行编辑、移动到已买/待买、删除
- 一键清空待买、清空已买、批量移动商品
- 本地保存清单数据

## 安全与隐私

- 不需要登录账号
- 不包含广告 SDK
- 不包含云同步或后台上传逻辑
- 不申请定位、相机、录音、通讯录、存储读取等敏感权限
- 购物清单数据保存在用户自己的设备本地
